# PWA Icons Setup

You need to add the following icon files to the `public` folder:

1. **pwa-192x192.png** - 192x192px icon
2. **pwa-512x512.png** - 512x512px icon  
3. **apple-touch-icon.png** - 180x180px icon for iOS
4. **mask-icon.svg** - SVG icon for Safari pinned tabs
5. **favicon.ico** - Standard favicon

## Quick Setup:

You can generate all these icons from a single source image using:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Temporary Solution:

For now, the PWA will work without icons but will show default placeholders.
